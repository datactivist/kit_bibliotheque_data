22/07/2020

Introduction

Ce kit vise à fournir aux bibliothèques souhaitant entamer une démarche d’extension des horaires d’ouverture les outils leur permettant de traiter les données disponibles pour documenter leur bilan.

Il se propose ainsi comme un complément du kit bibliothèque proposé par l’Agence régionale du livre Provence Alpes Côtes d’Azur. A cet égard, il s’attardera sur trois étapes que les données disponibles permettent de documenter de manière efficace :

  1. Le diagnostic de l’établissement ;

  2. Le diagnostic du territoire ;

  3. Les usagers et non usagers.

Pour chaque partie, une fois les enjeux de données posées, nous exposerons :

  • la méthode de collecte des données ;

  • un outil gratuit de visualisation ;

  • des exemples pratiques de traitements et visualisations à intégrer à votre projet.

Ce kit est open source, il s’appuie sur le travail réalisé par Datactivist pour l’Agence régionale du livre Provence Alpes Côtes d’Azur.
